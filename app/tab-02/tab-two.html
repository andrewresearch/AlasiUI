This app, written by Andrew Gibson, is one of a suite of apps (including GoingOK), and is part of the realtime component of a Lambda Architecture

Server-side code is a REST API written in Scala using Finagle/Finch, Circe, and Factorie libraries

Client-side is written in html and javascript using AngularJS, Bootstrap, and d3.js

This webapp is open source. The API code is available at ReflectivePhrases (GitHub) and the client code is available at AlasiUI (GitHub) . Other software is available from my GitHub (AndrewResearch) page.
